# blog_portfolio

react, js 사용

------------------------------------------------------------------

- react-router-dom 사용하여 페이지 이동.
- faker.js를 사용하여 더미데이터 구성.
- redux, saga, immer를 사용하여 전역 상태 관리.

------------------------------------------------------------------

- antDesign, styled-components, dayjs, shortid 라이브러리 사용.

------------------------------------------------------------------

- 게시글 및 댓글 구현.
- 회원가입 구현 : 이메일에는 @가 포함되어야 하며 비밀번호는 여섯 자리 이상이어야 한다. 이 조건을 충족할 시 버튼 활성화.
